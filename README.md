# CodeBanana Desktop Application

A Windows desktop application that provides all CodeBanana functionality without requiring a web browser.

## Features
- Full CodeBanana AI-powered development platform
- Direct desktop execution
- No browser required
- Windows executable (.exe) format

## How to Run

### Method 1: Batch File (Recommended)
1. Double-click `run.bat`
2. The application will start automatically

## Installation

1. Extract the application folder
2. Run `run.bat` to start the application

## Technical Details
- Built with Electron for cross-platform compatibility
- Uses Next.js for the user interface
- Packaged as Windows executable

## System Requirements
- Windows 7 or later
- Node.js installed
- 2GB RAM minimum

## Build Process
The application is structured to:
- Run locally without internet connection
- Provide full AI code generation capabilities
- Maintain all existing CodeBanana features

## Development
To modify the application:
1. Edit the HTML/CSS/JS files
2. Update Electron configuration as needed
3. Rebuild for changes
